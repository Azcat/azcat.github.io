为什么百度只收录了我的首页？

2019-05-14

Why 为什么会有这篇文章呢？

最近再试着做公司官网的SEO优化，各种刷都用上了，虽然排名有所上升，但是百度却只收录网站的首页，而且印象中首页的链接也是自己提交的。 😢

What 什么是提交百度链接呢？

首先要明白一个新站是非常不容易被百度的爬虫爬取的，所以这就需要我们将网站页面的链接通过百度站长工具手动提交给百度搜索。这样我们就能加快百度对网站的收录爬取。

How 怎样提交百度链接呢？

百度站长工具给出以下几种方式

  • 自动提交
    1. 主动推送
    2. 自动推送
    3. sitemap
  • 手动提交
    1. 普通数据
    2. 新站数据
      具体说明请参考百度站长工具

比较懒，比较懒，比较懒，重要原因说三遍。

我选择了自动推送，简单来说就是将百度提供的代码放到网站的模版中，这样每当有人访问过我的新页面，脚本就会自动提交链接。😁

代码：

<script>
(function(){
    var bp = document.createElement('script');
    var curProtocol = window.location.protocol.split(':')[0];
    if (curProtocol === 'https') {
        bp.src = 'https://zz.bdstatic.com/linksubmit/push.js';
    }
    else {
        bp.src = 'http://push.zhanzhang.baidu.com/push.js';
    }
    var s = document.getElementsByTagName("script")[0];
    s.parentNode.insertBefore(bp, s);
})();
</script>
